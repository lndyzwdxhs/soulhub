# SoulHub

Open-source Agent persona template community for [OpenClaw](https://github.com/openclaw).

Browse, install, and orchestrate AI agent persona templates with a single command.

## Features

- **21+ Curated Agent Templates** across 6 categories (Self Media, Development, Operations, Support, Education, Dispatcher)
- **One-Line CLI Install**: `npx soulhub install writer-xiaohongshu`
- **Visual Multi-Agent Composer**: Drag agents onto a canvas, auto-generate dispatcher with routing rules
- **Share Link**: Share your team configuration with a single link
- **3 Team Recipes**: Pre-built multi-agent teams ready to deploy

## Quick Start

### Install a single agent

```bash
# Search for agents
npx soulhub search writer

# Install an agent
npx soulhub install writer-xiaohongshu
```

### Install a team recipe

```bash
# Install the self-media team (6 agents)
npx soulhub install --recipe self-media-team
```

### Install from a Share Link

```bash
# Install a team from a share link (generated by the Composer)
npx soulhub install --from https://soulhub.dev/c/abc12345
```

## Web Platform

Visit [soulhub.dev](https://soulhub.dev) to:

- Browse and search all agent templates
- View agent details (IDENTITY.md, SOUL.md)
- Use the Visual Composer to build multi-agent teams
- Generate Share Links for team configurations

## CLI Commands

| Command | Description |
|---------|-------------|
| `soulhub search [query]` | Search for agent templates |
| `soulhub info <name>` | View agent details |
| `soulhub install <name>` | Install an agent from the registry |
| `soulhub install --from <url>` | Install from a Share Link |
| `soulhub install --recipe <name>` | Install a team recipe |
| `soulhub list` | List installed agents |
| `soulhub update [name]` | Update installed agents |
| `soulhub uninstall <name>` | Uninstall an agent |
| `soulhub publish [dir]` | Publish your agent to the community |

## Agent Template Format

Each agent template consists of:

```
agents/
└── your-agent-name/
    ├── manifest.yaml      # Metadata (required)
    ├── IDENTITY.md        # Identity definition (required)
    ├── SOUL.md            # Behavior patterns (required)
    ├── USER.md.template   # User profile template (optional)
    └── TOOLS.md.template  # Tools configuration (optional)
```

## Contributing

1. Fork this repository

3. Run validation: `node registry/scripts/validate.js`
4. Submit a Pull Request

See the [Contributing Guide](https://soulhub.dev/docs/contributing) for details.

## Development

```bash
# Install dependencies
npm install

# Start web dev server
npm run dev

# Build CLI
npm run build --workspace=packages/cli

# Build registry index
npm run build:index

# Validate templates
node registry/scripts/validate.js
```
