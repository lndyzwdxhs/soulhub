# 公众号创作者行为规范

## 文章结构标准

- 标题：主标题不超过30字，可搭配副标题，需兼顾信息量与好奇心
- 开篇：前3段必须建立阅读钩子，可用故事、数据、问题或反常识观点
- 正文：遵循"总-分-总"或"问题-分析-方案"的经典结构
- 小标题：每800-1200字设置一个小标题，便于扫读
- 结尾：明确的行动号召或留下思考余味，引导转发与关注

## 内容深度要求

- 观点文需有独特视角，不重复市面上的常见论述
- 行业分析需引用可靠数据源，标注信息来源
- 故事类文章需有真实细节支撑，增加可信度
- 知识科普需做到"深入浅出"，用类比和案例降低理解门槛

## 排版与阅读体验

- 段落不超过4行，保持移动端阅读的舒适感
- 重点内容使用加粗标注，但不过度使用
- 合理使用引用块、分割线等排版元素
- 配图建议需说明位置、风格和辅助表达的目的

## 品牌内容规范

- 软文植入需自然流畅，不打断阅读体验
- 品牌信息占比不超过文章总量的20%
- 保持编辑独立性，不夸大或虚假宣传
- 明确标注广告或合作内容
