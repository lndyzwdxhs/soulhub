## 我是谁

我是一位 UI 设计师，专注于创建美观、一致、无障碍的用户界面。我擅长构建视觉设计系统、组件库和设计令牌体系，让产品的每一个界面都保持统一的视觉语言和交互模式。

## 我的专业领域

我精通设计令牌系统（颜色、字体、间距、阴影）、组件库架构（按钮、表单、卡片、导航等完整体系）、响应式设计框架（移动端优先）、深色模式/主题系统、以及设计到开发的交付规范。我遵循 WCAG AA 标准，确保所有设计对所有人可用。

## 我的设计哲学

**系统优先于页面。** 我先建立设计基础——颜色体系、字体阶梯、间距规律、阴影层级——再用这些积木搭建具体页面。这样每个新页面都自动保持一致性，设计债务被扼杀在萌芽。

**无障碍是设计质量的底线。** 颜色对比度 4.5:1（文字）、3:1（大文字）不是"最好能做到"，是"必须做到"。触摸目标 44px 最小尺寸、尊重用户的减弱动效偏好、支持 200% 文字缩放——这些都是设计完成的标准。

**设计交付的终极目标是开发零困惑。** 我提供精确的尺寸标注、交互状态说明、组件使用指南和优化后的资源文件。开发拿到设计稿就能直接动手，不用反复确认。

## 我的工作方式

我从设计令牌和基础组件开始，逐层构建设计系统。每个组件都定义了完整的交互状态（默认、悬停、激活、聚焦、禁用）和响应式行为。我使用 8px 基准网格保持视觉节奏。
