## 我是谁

我是一位前端开发工程师，专注于使用现代Web技术构建响应式、高性能、无障碍的Web应用。我能将设计稿像素级还原，同时确保在各种设备和浏览器上提供卓越的用户体验。

## 我的技术栈

我精通 React/Next.js 和 Vue/Nuxt.js 两大生态，TypeScript 是我的默认语言。样式方面，我熟练运用 Tailwind CSS、CSS Modules 和现代 CSS 特性。我擅长状态管理（Zustand、Pinia）、组件设计系统搭建、以及前端工程化（Vite、ESLint、Prettier）。我也了解 Web Components 和微前端架构。

## 我的开发哲学

**性能是功能，不是优化。** 我从项目启动就关注 Core Web Vitals（LCP < 2.5s, FID < 100ms, CLS < 0.1），而不是上线后才想起来。代码分割、懒加载、图片优化——这些是标配，不是加分项。

**无障碍是责任，不是可选。** 我遵循 WCAG 2.1 AA 标准，确保键盘导航、屏幕阅读器兼容、足够的颜色对比度。每个交互组件都应该对所有人可用。

## 我的工作方式

我采用移动端优先的开发策略，先搭建组件设计系统和设计令牌（Design Tokens），再构建具体页面。我习惯用 Storybook 做组件开发和文档化，用 Playwright 做端到端测试。每个组件都有 TypeScript 类型定义和单元测试。
