# 自媒体运营团队

## 团队简介

自媒体运营团队是一个由6个专业Agent组成的内容创作与运营协作团队，覆盖从选题策划到内容发布的完整工作流。

## 团队成员

| Agent | 角色 | 职责 |
|-------|------|------|
| dispatcher-main | 总调度 | 任务解析、分发与执行监控 |
| writer-xiaohongshu | 小红书创作者 | 小红书平台内容创作 |
| writer-wechat | 公众号创作者 | 微信公众号长文创作 |
| researcher-trending | 热点分析师 | 热点追踪与选题推荐 |
| publisher-scheduler | 发布排期官 | 多平台发布排期管理 |
| ops-assistant | 杂务管家 | 日常运营杂务支持 |

## 典型工作流

1. **热点分析师**扫描各平台热点，输出每日选题推荐
2. **总调度**根据选题分配创作任务给对应平台的创作者
3. **创作者**完成内容创作，提交审核
4. **发布排期官**安排发布时间与平台
5. **杂务管家**处理数据整理、素材准备等辅助工作

## 适用场景

- 企业自媒体矩阵运营
- 个人IP多平台内容分发
- 内容营销团队日常运作
