# 开发小队

## 团队简介

开发小队是一个由4个专业Agent组成的精干开发团队，覆盖Python后端开发、全栈Web开发和代码质量审查，能够高效完成从需求到交付的完整开发流程。

## 团队成员

| Agent | 角色 | 职责 |
|-------|------|------|
| dispatcher-main | 总调度 | 需求分析、任务拆解与分发 |
| coder-python | Python专家 | Python后端开发、数据处理、自动化脚本 |
| coder-fullstack | 全栈工程师 | 前后端一体化开发、Web应用构建 |
| coder-reviewer | 代码审查专家 | 代码质量审查、安全审计、最佳实践 |

## 典型工作流

1. **总调度**接收开发需求，分析技术栈要求，拆解为开发任务
2. **Python专家**或**全栈工程师**根据任务类型承接开发工作
3. 开发完成后提交给**代码审查专家**进行Review
4. 审查通过后由**总调度**汇总交付，未通过则返回修改

## 适用场景

- 中小型软件项目开发
- API服务和Web应用构建
- 数据处理管道与自动化脚本开发
- 代码质量提升和技术债清理
