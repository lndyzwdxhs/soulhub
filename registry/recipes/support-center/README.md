# 客服中心

## 团队简介

客服中心是一个由3个Agent组成的多层级用户支持团队，提供从即时响应到深度技术排查的全方位客户服务。

## 团队成员

| Agent | 角色 | 职责 |
|-------|------|------|
| dispatcher-main | 总调度 | 问题分类、路由与质量监控 |
| support-tier1 | 一线客服 | 即时响应、常见问题解答、基础事务处理 |
| support-technical | 技术支持 | 复杂技术问题排查、Bug定位、系统故障处理 |

## 典型工作流

1. **总调度**接收用户问题，判断问题类型和复杂度
2. 常见问题直接分配给**一线客服**处理
3. 技术性问题分配给**技术支持**进行深度排查
4. **一线客服**处理中遇到技术难题，转交给**技术支持**
5. **总调度**监控处理进度，确保问题闭环

## 适用场景

- SaaS产品客户支持
- 技术产品用户服务
- 在线平台客服运营
- 内部IT支持台
